# dsh-select

Web component de selector del DSH Design System. Construido con Atomico sobre Shadow DOM.

Ofrece un selector personalizable con dos modos: **selección única** (`type="select"`) y **selección múltiple** (`type="checkbox"`). Las opciones se definen mediante el componente hijo `<dsh-select-option>`. Soporta modo **modal** para mejor experiencia en dispositivos móviles y ordenamiento **alfabético** automático.

## dsh-select-option

El componente `<dsh-select-option>` define cada opción del selector:

| Prop | Tipo | Descripción |
|------|------|-------------|
| `value` | string | Valor enviado en los eventos |
| `selected` | boolean | Opción preseleccionada al renderizar |
| `disabled` | boolean | Deshabilita la opción individualmente |
| `error` | boolean | Aplica estilo de error a la opción |

## Eventos

| Evento | Detail | Descripción |
|--------|--------|-------------|
| `ClickOption` | value (string) | Se dispara al seleccionar una opción (type="select") |
| `clickCheckbox` | values (string[]) | Se dispara al cambiar la selección múltiple (type="checkbox") |
| `Selected` | — | Se dispara al confirmar la selección |
