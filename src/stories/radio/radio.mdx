# dsh-radio

Web component de radio button del DSH Design System. Construido con Atomico sobre Shadow DOM.

Permite la selección de una única opción dentro de un grupo. Incluye el componente complementario `<dsh-radio-group>` para gestionar la selección exclusiva automáticamente. Soporta estados **default**, **checked**, **disabled**, **error** y **dark mode**.

|--------|-------------|
| `onClick` | Se dispara al seleccionar el radio button |

## React — Cómo usarlo

El componente `<dsh-radio>` emite el evento `onClick` al ser seleccionado. Para múltiples opciones, envuélvelos en `<dsh-radio-group>`.

<Canvas of={RadioStories.ReactUsage} />

El tab **Show code** muestra los dos patrones de consumo:
- **React 19** — prop `onclick` directamente en JSX
- **React 18** — `useRef` + `addEventListener('onClick', handler)`
