# dsh-input

Web component de campo de texto del DSH Design System. Construido con Atomico sobre Shadow DOM.

Ofrece un campo de entrada de texto completo con soporte para múltiples tipos, estados visuales, mensajes de ayuda y error, limpieza del valor y accesibilidad completa. Incluye funcionalidad especial para el tipo `rut` con validación de RUT chileno.

## React — Cómo usarlo

El componente `<dsh-input>` emite el evento `changeInput` con el nuevo valor como `event.detail`.

<Canvas of={InputStories.ReactUsage} />

El tab **Show code** muestra los dos patrones de consumo:
- **React 19** — prop `onchangeinput` directamente en JSX
- **React 18** — `useRef` + `addEventListener('changeInput', handler)`
