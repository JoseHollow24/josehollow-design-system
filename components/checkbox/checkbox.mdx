# dsh-checkbox

Web component de checkbox del DSH Design System. Construido con Atomico sobre Shadow DOM.

Ofrece control de selección binaria con soporte para todos los estados de formulario: **default**, **checked**, **disabled**, **error**, **indeterminate** y **dark mode**. Emite eventos personalizados para cambios de estado, foco y blur.

## React — Cómo usarlo

El componente `<dsh-checkbox>` emite el evento `checkbox-change` con los datos del campo en `event.detail`.

<Canvas of={CheckboxStories.ReactUsage} />

El tab **Show code** muestra los dos patrones de consumo:
- **React 19** — prop `oncheckbox-change` directamente en JSX
- **React 18** — `useRef` + `addEventListener('checkbox-change', handler)`
