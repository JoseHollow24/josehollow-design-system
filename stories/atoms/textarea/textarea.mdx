# dsh-textarea

Web component de área de texto del DSH Design System. Construido con Atomico sobre Shadow DOM.

Proporciona un campo de texto multilínea con contador de caracteres, mensaje de ayuda, estados visuales de error/éxito, redimensionamiento automático y control de longitud máxima. Emite el evento `changeTextarea` con el contenido actual al escribir.

## React — Cómo usarlo

El componente `<dsh-textarea>` emite el evento `changeTextarea` con el nuevo valor como `event.detail`.

<Canvas of={TextareaStories.ReactUsage} />

El tab **Show code** muestra los dos patrones de consumo:
- **React 19** — prop `onchangetextarea` directamente en JSX
- **React 18** — `useRef` + `addEventListener('changeTextarea', handler)`
