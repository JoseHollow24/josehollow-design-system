# dsh-icon

Web component de ícono del DSH Design System. Construido con Atomico sobre Shadow DOM.

Integra íconos de **FontAwesome** con soporte para los tokens de color y tamaño del Design System. Permite seleccionar la familia del ícono (`fas`, `far`, `fal`, `fab`), el nombre del ícono (`fa-bell`, `fa-user`, etc.), el tamaño con escala `s1`–`s7` y el color usando tokens del Design System.

## Tamaños

La escala de tamaños va de `s1` (más pequeño) a `s7` (más grande).
