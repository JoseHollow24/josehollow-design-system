# dsh-tag

Web component de etiqueta del DSH Design System. Construido con Atomico sobre Shadow DOM.

Permite comunicar estados, categorías y contextos mediante etiquetas visuales. Soporta cinco variantes principales (**general**, **informative**, **news**, **menu**, **interactive**) y cuatro subvariantes de estado (**success**, **error**, **warning**, **informative**) con soporte opcional de íconos.

## Informative — Subvariantes

La variante `informative` admite cuatro subvariantes para comunicar estados semánticos. Cada una puede mostrar un ícono representativo activando `show-icon`.
